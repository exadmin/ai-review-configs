# ai-review-configs
Centralized configurations storage for the AI-Review tool


## How to setup your repository
Add file "ai-review-deepseek-centralized.yml" to your repository into ".github/workflows/*.yml" folder and commit/push changes.

Setup GitHub secrets:
OPENAI_API_KEY - Deepseek token, can be requested by https://platform.deepseek.com/
PR_RW_TOKEN - GitHub token with read/write access to Pull Requests
